Operators

R arithmetic operators are + - * / ^ % any %. For example:

a %/% b  # Intiger division
a %% a   # Remainder
A %*% B  # Matrix multiplication
a ^ n    # nth power 
a ^ 1/n  # nth root

The other operators include:

  • Negation: !
  • Indexing: [ [[
  • Sequence operator: :
  • Component/slot extraction: $ @
  • Logical: & && (and) | || (or)
  • Membership: %in%
  • Assignment: = <- ->
  • Ordering and comparison: < > <= >= == !=

Data structures

There are five major data structure functions in R:

  • c
  • cbind (rbind)
  • matrix
  • data.frame
  • list

c combines values into a vector. cbind (rbind) takes a sequence of vector, matrix or data-frame arguments and combine by columns (rows), respectively. This function keeps all array with a same type (e.g all int or char). matrix creates a matrix from the given set of values. In matrix we can use [ , ] to call columns and rows.

data.frame creates data frames, store each column separately as a different variable with different observations (n obs. of m variables). When we read a csv file it saves as a dataframe. In data frames we can use $ to call columns and [ , ] for calling a certain index.

list makes a data structure that each row is not necessarily equal. For example if you have a loop that do not generate same amount of results at each iteration then you can store them in a list format. In lists we can use $ or [[ ]] to call components and [ ] to call members of each component.

Matrix operations

matrix function creates a matrix from the given set of values. For example matrix(c(1,2,3,3,2,1), 2, 3) generates a \(\small 2 \times 3\) matrix such as:

##      [,1] [,2] [,3]
## [1,]    1    3    2
## [2,]    2    3    1

The most common matrix operations include:

  • Dimensions of a matrix: dim()
  • Diagonal of a matrix: diag()
  • Identity matrix: diag(number)
  • Matrix symmetric test: isSymmetric()
  • Matrix multiplication: %*%
  • Test matrices for exact equality: identical(,)
  • Matrix transpose: t()
  • Determinant of a matrix: det()
  • Inverse of a matrix: solve()
  • Solve a system of equations: solve()
  • QR Decomposition of a Matrix: qr()
  • Spectral Decomposition of a Matrix: eigen()

The following are some examples of above functions.

Split and merge data

split function divides the data into the defined groups. merge function merge two data frames by common columns or row names. merge is similar to JOIN in SQL:

  • Inner join: merge(x, y, by)
  • Left join: merge(x, y, by, all.x = T)
  • Right join: merge(x, y, by, all.y = T)
  • Outer join: merge(x, y, by, all = T)

Regular expressions

Regular expressions (regex) are a simple way to find patterns in text. The most common functions implementing regex in R are in the global regular expression print (grep) family. For more information see help("regex") and help(grep). Also, You may learn more about regex functions in R at D-RUG. Following are some simple examples.

sub("a", "#", "abcdfa") # Substitution: substitue first "a" in the expression "abcdfa" with "#"

gsub("a", "#", "abcdfa") # General substitution: substitue all "a" in the expression "abcdfa" with "#"  
## [1] "#bcdf#"
gsub("//#", "a", "//#bcdf//#") # For "$","*","+",".","?","[","^","{","|","(", and "\\" need to add "//" before!

grep("a", c("abc", "def", "cba a", "aa"), perl = TRUE, value = FALSE) # Shows arrays' number that include "a"
## [1] 1 3 4
grep("a+", c("abc", "def", "cba a", "aa"), perl = TRUE, value = TRUE) # Shows expressions that include "a"
## [1] "abc"   "cba a" "aa"
substr("abcdef", 1, 3) # substr(x, start, stop)

Defining functions

In R we can define any function that we need. It’s very useful since we need to do a certain calculation frequently. Defining functions, also, let us use recursive algorithm which is a powerful algorithm to solve the problems. The following show some examples.

norm <- function(x) {
  sqrt(x%*%x)

# Recersive function that compute n!
fact <- function(n) { 
  if (n == 1) 
    1
  else 
    n * fact(n-1) # Recursive function
}

# Recersive function that compute matrix power
matrix.power <- function(p, n) { 
  if (n == 1) 
    p
  else 
    p %*% matrix.power(p, n-1) # Recursive function

    if (identical(a, t(a)) == TRUE) {
      print("Matrix is symmetric")
    } else print("Matrix is not symmetric")
  } else print("Entery is not a Matrix")
}
